## UI Experience:

### User login with API key

**Parameters:** 

API key: (query, string, required): The API key to access backend service.

username: (query, string, required): The username to log in

### Associate Account

**Parameters:**

oauth credentials: (query, string, required): The credentials used to associate the email account with the username.

### Chat

Click on chat box to enable chat and interact with the agent through the UI. 

### Dissociate Account

Click on dissociate account button to delink the account with the username.
